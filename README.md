This repository contains the official Flutter mobile application for ParkChain, a peer-to-peer parking marketplace built on Web3 technologies.

This app is targeted at **Drivers**, allowing them to find, book, and pay for parking spots using cryptocurrency.

## Project Overview

* **Web3 Authentication:** Login via Web3Auth using crypto wallets.
* **Map-Based Search:** Find and filter nearby parking spots in real-time.
* **Crypto Payments:** Secure, low-fee booking payments.
* **GPS Verification:** Real-world validation of driver arrival (check-in) and departure (check-out) using GPS fencing.
* **Immutable Reviews:** On-chain reviews for transparent, trustworthy feedback.
